# Sampurasun-1-app
## Deskripsi
Aplikasi web untuk manajemen pelaporan stakeholder dengan dashboard admin dan fitur PWA.

## Fitur
- Dashboard stakeholder
- Dashboard admin
- Progressive Web App (PWA)

## Cara Menjalankan Aplikasi
1. Clone repository ini.
2. Instal dependensi: `npm install`.
3. Jalankan aplikasi: `npm start`.

## Deployment
- Frontend: Deploy ke Vercel.
- Backend: Deploy ke Railway.
